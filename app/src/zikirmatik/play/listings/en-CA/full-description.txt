Zikirmatik – advanced tasbih/dhikr counter. Designed for fast, error-free use with multi-profile support, target progression and precise touches. No internet required; The data is kept on the device.

FEATURES
- Multiple counters: name, target (optional), description, color/icon, praise mode.
- Quick increase/decrease/reset: large touch areas for one-handed access.
- Target progress: percentage, remaining, number of rounds (33/99 steps in praise mode).
- Haptics: short vibration with each step, long vibration on target/lap completion.
- Keep screen on (optional): for long sessions.
- Undo: fix wrong touches with Undo within 5 seconds.
- History and statistics: session record with timestamp; daily/weekly summaries.
- Reminders: daily/weekly notification; Auto skip during quiet hours.
- Theme: dark/light/system.
- Advertisements: banner/native advertisement display within the application.
- Privacy: no accounts, no tracking; The data is stored in the local database (Room).

USAGE TIPS
- When tasbih mode is on, the rotation increases with every 33 steps and gives a long vibration.
- The percentage/remainder is automatically calculated in the counters where you set a target.
- “Keep screen on” is applied only on the meter detail screen.
- Access reduce/reset/edit/archive operations from long press menu.
- Manage reminder time/period and quiet time interval from Settings.

PRIVACY
- Data is stored on the device; No personal data, location or contact information is collected.
- Internet connection is used only to load ads.

FEEDBACK
- You can send feedback for errors/suggestions via store evaluation.
