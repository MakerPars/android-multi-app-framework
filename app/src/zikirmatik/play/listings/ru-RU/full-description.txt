Зикирматик – продвинутый счетчик тасбиха/зикра. Разработан для быстрого и безошибочного использования с поддержкой нескольких профилей, прогрессом цели и точными касаниями. Интернет не требуется; Данные сохраняются на устройстве.

ОСОБЕННОСТИ
- Несколько счетчиков: имя, цель (необязательно), описание, цвет/значок, режим похвалы.
- Быстрое увеличение/уменьшение/сброс: большие сенсорные области для доступа одной рукой.
- Целевой прогресс: процент, остаток, количество раундов (33/99 шагов в режиме похвалы).
- Тактильные ощущения: короткая вибрация при каждом шаге, длинная вибрация при достижении цели/круга.
- Оставлять экран включенным (необязательно): для длительных сеансов.
- Отменить: исправьте неправильные касания с помощью функции «Отменить» в течение 5 секунд.
- История и статистика: запись сеанса с отметкой времени; ежедневные/еженедельные сводки.
- Напоминания: ежедневное/еженедельное уведомление; Автоматический пропуск в тихие часы.
- Тема: темная/светлая/система.
- Рекламные объявления: отображение баннеров/родной рекламы в приложении.
- Конфиденциальность: нет учетных записей, нет отслеживания; Данные хранятся в локальной базе данных (Room).

СОВЕТЫ ПО ИСПОЛЬЗОВАНИЮ
- При включенном режиме тасбих вращение увеличивается каждые 33 шага и дает продолжительную вибрацию.
- Процент/остаток автоматически рассчитывается в счетчиках, где вы устанавливаете цель.
- «Не отключать экран» применяется только на экране сведений о счетчике.
- Доступ к операциям уменьшения/сброса/редактирования/архивирования из меню длительного нажатия.
- Управляйте временем/периодом напоминания и интервалом тихого времени в настройках.

КОНФИДЕНЦИАЛЬНОСТЬ
- Данные хранятся на устройстве; Никакие личные данные, местоположение или контактная информация не собираются.
- Интернет-соединение используется только для загрузки рекламы.

ОБРАТНАЯ СВЯЗЬ
