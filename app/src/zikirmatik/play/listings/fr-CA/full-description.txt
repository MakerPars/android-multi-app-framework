Zikirmatik – compteur tasbih/dhikr avancé. Conçu pour une utilisation rapide et sans erreur avec une prise en charge multi-profils, une progression cible et des touches précises. Aucune connexion Internet requise ; Les données sont conservées sur l'appareil.

CARACTÉRISTIQUES
- Plusieurs compteurs : nom, cible (facultatif), description, couleur/icône, mode d'éloge.
- Augmentation/diminution/réinitialisation rapide : grandes zones tactiles pour un accès d'une seule main.
- Progression cible : pourcentage, reste, nombre de tours (33/99 étapes en mode éloge).
- Haptique : vibration courte à chaque pas, vibration longue à la fin de la cible/du tour.
- Garder l'écran allumé (facultatif) : pour les longues sessions.
- Annuler : corrigez les mauvaises touches avec Annuler dans les 5 secondes.
- Historique et statistiques : enregistrement de session avec horodatage ; résumés quotidiens/hebdomadaires.
- Rappels : notification quotidienne/hebdomadaire ; Saut automatique pendant les heures calmes.
- Thème : sombre/lumière/système.
- Publicités : affichage de bannières/publicités natives au sein de l'application.
- Confidentialité : pas de compte, pas de suivi ; Les données sont stockées dans la base de données locale (Room).

CONSEILS D'UTILISATION
- Lorsque le mode tasbih est activé, la rotation augmente tous les 33 pas et donne une longue vibration.
- Le pourcentage/reste est calculé automatiquement dans les compteurs où vous avez fixé un objectif.
- « Garder l'écran allumé » s'applique uniquement sur l'écran de détail du compteur.
- Accédez aux opérations de réduction/réinitialisation/édition/archivage à partir du menu d'appui long.
- Gérez l'heure/la période de rappel et l'intervalle de temps silencieux à partir des paramètres.

CONFIDENTIALITÉ
- Les données sont stockées sur l'appareil ; Aucune donnée personnelle, localisation ou information de contact n’est collectée.
- La connexion Internet est utilisée uniquement pour charger des annonces.
